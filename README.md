# Wells Fargo (wells-fargo)
Wells Fargo & Company is one of the largest banks in the United States. Wells Fargo operates the Gateway developer portal at developer.wellsfargo.com, one of the first open banking platforms launched in 2016, offering a growing catalog of plug-and-play APIs, SDKs, and webhooks for commercial customers and partners across payments, account data, and financial services.

## APIs

### Gateway API
The Wells Fargo Gateway API is a comprehensive open banking platform offering plug-and-play APIs, SDKs, and webhooks for commercial customers and partners. The platform supports payments, real-time account information, and financial data integration across commercial banking, mortgage, credit card, and brokerage services. Built on RESTful principles with OAuth 2.0 authentication, handling over 1.5 billion API calls annually.
